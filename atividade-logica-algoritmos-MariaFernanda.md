# Atividade Prática de Lógica e Algoritmos

## Controle de Acesso

Início

lista_oficial = lista de alunos permitidos  
fila = alunos aguardando entrada  

Para cada aluno na fila faça  
    Se aluno está na lista_oficial então  
        Exibir "Entrada permitida"  
    Senão  
        Exibir "Entrada negada - aluno não encontrado na lista"  
    FimSe  
FimPara  

Fim
